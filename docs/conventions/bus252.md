# Convenção bus/252 — Base Day Count do Mercado Brasileiro

## Definição

A convenção **bus/252** (também chamada de *dias úteis sobre 252*) é o padrão de contagem de dias para a grande maioria dos instrumentos denominados em BRL no mercado brasileiro. Diferentemente dos mercados internacionais (que tipicamente usam ACT/360 ou ACT/365), o Brasil usa **dias úteis** no numerador e **252** (número convencional de dias úteis por ano) no denominador.

> **Referência normativa:** ANBIMA — Manual de Marcação a Mercado, Seção 2 (Convenções de Mercado). B3 — Especificações Gerais de Contratos.

## Fórmula

```
fator = (1 + taxa_anual) ^ (dias_uteis / 252)
```

Onde `dias_uteis` é o número de dias úteis entre a data de liquidação (inclusive) e a data de vencimento (exclusive), contados pelo **calendário ANBIMA**.

## Quando usar bus/252

| Instrumento | Convenção |
|-------------|-----------|
| LTN, NTN-F, NTN-B, LFT, NTN-C | bus/252 |
| CDB, LCI, LCA, LF, DPGE | bus/252 |
| Debêntures (pré, DI%) | bus/252 |
| Futuro DI (DI1) | bus/252 |
| Swap pré × DI | bus/252 (perna pré) |
| Compromissadas | bus/252 |

## Quando NÃO usar bus/252

| Instrumento | Convenção correta |
|-------------|-------------------|
| Títulos em USD (eurobonds) | ACT/360 ou ACT/365 |
| Cupom cambial (DDI) — perna USD | ACT/360 |
| NDF liquidado em USD | ACT/360 |
| Produtos indexados a LIBOR/SOFR | ACT/360 |

## Impacto prático

O uso de dias úteis (não corridos) faz com que o **fator de acumulação seja zero em feriados** — a taxa não cresce em fins de semana ou feriados para instrumentos bus/252. Isso é fundamentalmente diferente de ACT/365, onde o tempo passa continuamente.

Exemplo: uma LTN com taxa de 12% a.a. entre duas datas com 5 dias úteis:
```
fator = (1,12) ^ (5/252) = 1,002246...
```

## Erros comuns

- Usar dias corridos em vez de dias úteis (erro mais comum)
- Usar 252 mas com calendário incorreto (sem feriados ANBIMA)
- Usar ACT/252 (corridos sobre 252) — não existe no mercado brasileiro
- Confundir com ACT/365 em contexto de comparação internacional
